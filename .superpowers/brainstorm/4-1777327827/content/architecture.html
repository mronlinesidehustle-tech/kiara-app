State Machine (inside ReadingLesson.jsx)

phase state
intro → sight-words → phonics → short-story → completion

Each phase renders its own UI within the same component. A phase state variable drives what's shown — same pattern as step in the math lessons.

Data Flow

  • Default word lists and story are hardcoded constants in ReadingLesson.jsx
  • Parent overrides are stored in localStorage (key: reading-config-{studentId}) and editable from ParentDashboard
  • On session start, ReadingLesson reads localStorage — falls back to defaults if nothing set
  • Completed session saved via saveProgress() (same KV sync as math lessons)
