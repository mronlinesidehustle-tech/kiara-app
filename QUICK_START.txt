───────────────────────────────────────────────────────────────────────
STEP 1: GITHUB (5 min)
───────────────────────────────────────────────────────────────────────

1. Go to: https://github.com/new
2. Name: kiara-app
3. Choose "Public" and "Add README"
4. Click "Create repository"
5. Copy the HTTPS URL it shows

Terminal:
  git clone [PASTE_URL_HERE]
  cd kiara-app

───────────────────────────────────────────────────────────────────────
STEP 2: ADD CODE (2 min)
───────────────────────────────────────────────────────────────────────

Copy all files from this folder into your kiara-app folder.

Terminal (still in kiara-app folder):
  git add .
  git commit -m "Initial: Kiara's app"
  git push origin main

✅ Code is now on GitHub!

───────────────────────────────────────────────────────────────────────
STEP 3: DEPLOY TO VERCEL (10 min)
───────────────────────────────────────────────────────────────────────

1. Go to: https://vercel.com
2. Log in with GitHub (if no account, create one)
3. Click "New Project"
4. Find "kiara-app" in the list
5. Click "Import"
6. Click "Deploy" (keep all defaults!)

Wait for green checkmark... ✅

You'll get a URL like: https://kiara-app-abc123.vercel.app

COPY THIS URL! ↑↑↑

───────────────────────────────────────────────────────────────────────
STEP 4: TEST ON ANDROID PHONE (5 min)
───────────────────────────────────────────────────────────────────────

1. Open Chrome on your Android
2. Paste the Vercel URL
3. Tap "Start Lesson"
4. Try one problem
5. Check "Parent Dashboard"

✅ If it works, you're LIVE!
